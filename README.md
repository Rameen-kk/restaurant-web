# Tasty Bites - Restaurant Website

A responsive multi-page restaurant website built with HTML, CSS, and JavaScript.

Live Demo:
https://rameen-kk.github.io/restaurant-web/index.html
### **Features**
- **4 Pages**: Home, Menu, About, Contact
- **Responsive Design**: Mobile-friendly with hamburger menu
- **Menu Filtering**: Filter dishes by category - Fast Food, BBQ, Pizza, Burgers, Drinks, Desserts
- **Contact Form**: Client-side validation with JavaScript
- **Back to Top Button**: Smooth scroll functionality
- **Google Maps**: Embedded location map
- **Hover Effects**: Cards and buttons with smooth animations
- **Sticky Navbar**: Navigation stays on top while scrolling

### **Tech Stack**
- **HTML5**: Semantic markup
- **CSS3**: Flexbox, Grid, Media Queries, Animations
- **Vanilla JavaScript**: DOM manipulation, form validation, filtering
- **No Frameworks**: Pure HTML/CSS/JS as per assignment requirements
